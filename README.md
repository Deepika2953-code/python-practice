# Python Practice

A collection of beginner-to-intermediate Python programs focused on logic building and problem solving.

---

## Projects Included

### Number Guess Game
A console-based game where:
- The computer randomly selects a number between 1 and 50
- The user has 5 attempts to guess it
- Hints are provided after each incorrect attempt

---

## Concepts Used

- random module
- while loop
- if-elif-else
- break statement
- user input handling

---

## Purpose

To strengthen Python fundamentals and improve algorithmic thinking.

---

## How to Run

1. Make sure Python is installed.
2. Clone the repository.
3. Run:

python number_guess_game.py
